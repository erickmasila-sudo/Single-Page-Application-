# Personal Projects To-Do List
A Website That keeps a list of the projects one has to do, can both add new ones or delete once they are done.

## Table of Contents
- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)
- [License](#license)

## Installation
1. Clone the repository:
```bash
 git clone https://github.com/erickmasila-sudo/Single-Page-Application-.git
```

2. Install dependencies:
```bash
npm install
 ```

## Usage
To be able to see the project in local host use
```bash
npm run dev
```
### Features
**1. Form** <br/>

**Adding a Project**  - Write the Title and Description and click the Add button to put a new project in the list,
   - ***Error Checks*** - The form checks if Both the Title and Descritiption are added before submitting

**2. Lists** <br/>

- **Displaying**- It shows the list of to-dos <br />

- **Search**- A Feature which filters through the list and gets projects whose titles are related to the search term <br/>

- **Remove Button**- A feature which deletes a to-do from the list

**3. Storage** - Project to-dos are saved in the local storage
